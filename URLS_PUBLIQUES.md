Pour déployer une nouvelle version :

```bash

git add .

# 3. Poussez sur GitHub
git push

# 4. Déployez sur Vercel

vercel --prod
```

## 🎯 Domaine Personnalisé (Optionnel)

Si vous voulez un domaine personnalisé comme `datamatch.com` :

1. Achetez un domaine (ex: sur Namecheap, GoDaddy)
2. Allez sur Vercel Dashboard → Settings → Domains
3. Ajoutez votre domaine personnalisé
4. Configurez les DNS selon les instructions Vercel

## 🔧 Stack Technique

### Frontend
- **Framework** : React 19
- **Language** : TypeScript
- **Build** : Vite
- **UI** : Tailwind CSS + shadcn/ui
- **Charts** : Recharts
- **Maps** : Leaflet
- **DnD** : @dnd-kit
- **Cache** : IndexedDB + LocalStorage

### Backend
- **Runtime** : Node.js
- **Framework** : Express 5
- **Language** : TypeScript
- **Excel** : ExcelJS + xlsx
- **Utils** : Lodash
- **Upload** : Multer
- **Compression** : zlib

### Performance
- **Cache** : Multi-niveaux (Mémoire + IndexedDB + LocalStorage)
- **Pagination** : Intelligente avec préchargement
- **Lazy Loading** : Intersection Observer
- **Compression** : gzip
- **Workers** : Traitement asynchrone
